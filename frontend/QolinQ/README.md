# Qolinq - Premium Creator Marketplace

Qolinq is a high-end digital ecosystem connecting premium brands with top-tier influencers. Built with a focus on experience, security, and performance.



- **Frontend**: React + Vite + Tailwind CSS + TanStack 
- **Backend**: Node.js + Express + MongoDB
- **Key Features**:
  - Real-time Dashboard with Analytics
  - Secure Escrow Payment System
  - Intelligent Fraud Detection
  - Seamless Brand-Influencer Messaging

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- Bun or npm/yarn
- MongoDB instance

### Frontend Setup

```bash
cd frontend/QolinQ
bun install
bun dev
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

## Tech Stack

- **UI/UX**: Custom design system with neon aesthetics and glassmorphism.
- **State Management**: React Context + React Query for server state.
- **Backend Architecture**: MVC pattern with robust middleware security.
